# Discord ChatGPT bot
Discord bot integrated with ChatGPT

### Requirements
* Python 3
* OpenAI API key
* Discord API key


## Technology
* Python 3
* Discord
* OpenAI ChatGPT


## Setup
##### Create virtual environment
```
python3 -m venv venv
```

##### Install depedencies
```
pip install -r requirements.txt
```

##### Configure and adapt to your needs `.env`
```
cp .env.example .env
```

##### Run the application
```
python3 main.py
```
